# quickco-server

Please build client folder from quickco-v2 angular project


To config your datasource, edit datasources.json

e.g.
`
{
  "db": {
    "name": "db",
    "connector": "memory"
  },
  "mysql": {
    "host": "localhost",
    "port": 3306,
    "database": "mydb",
    "username": "root",
    "password": "111",
    "name": "db",
    "connector": "mysql"
  }
}
`
